## 1. Overview
Vision. Activities in the Triad is a platform that will allow customers to find local entertainment events and activities. The platform allows businesses to post events and activities at all price points for customers to find and book. Providers will utilize activity tags to attract customers interested in the type of event they are offering. Customers will set up a profile with activities preset so they can be offered events that match their interests.
